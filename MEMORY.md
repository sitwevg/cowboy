# Cowboy — рабочая память

## Статус
Активная разработка. Expo-приложение запускается на телефоне через Expo Go.

## Окружение
- Node.js через nvm, версия v20.20.0
- Активация: `source ~/.nvm/nvm.sh && nvm use v20.20.0`
- Expo SDK 54, совместим с Expo Go v54.0.6 (Android)
- Установка зависимостей: `npm install --legacy-peer-deps`
- Запуск: `cd ~/Z/cowboy/app && npx expo start`
- Тест на телефоне: сканировать QR в Expo Go (одна WiFi сеть)
- Тоннель: `npx expo start --tunnel` (если разные сети, ngrok установлен)

## Оригинальная игра
- Файлы в `original/`: WAY.EXE, USER.SAV, скриншоты, game_data.md
- Запуск в DOSBox: лаунчер на рабочем столе → "WAY: Ковбой"
- Скриншоты DOSBox → ~/Изображения/ → перемещать в original/screenshots/
- Прочитанные скриншоты → original/screenshots/done/
