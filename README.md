# В поисках сокровища — мобильный порт

Бесплатная адаптация DOS-игры **WAY.EXE «Ковбой»** (версия 2.21 beta, 19.08.1999) для мобильных устройств и браузера.

## Об оригинале

Оригинальная игра создана **Яценко Сергеем**, 1999 год. Текстовая RPG в жанре вестерн: ковбой ищет клад в пустыне, сражается с бандитами и волками, торгует с индейцами и в салунах, прокачивает навыки. Написана на Pascal, около 4400 строк кода.

## Об адаптации

Порт сохраняет дух оригинала:
- DOS-эстетика: чёрный фон, моноширинный шрифт, ASCII-рамки
- Оригинальные фразы и механики (включая намеренные баги из оригинала)
- Управление адаптировано под тачскрин вместо клавиатуры

Платформы: **Android** (Expo Go / APK) и **веб-браузер** (Vercel).

## Стек

- React Native + Expo SDK 54
- Expo Web (Metro bundler)
- JavaScript

## Запуск

```bash
cd app
npm install --legacy-peer-deps
npx expo start          # мобильный (Expo Go)
npx expo start --web    # браузер
```
